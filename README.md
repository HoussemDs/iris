## ğŸ“ Measurements Guide

For best predictions, measure your Iris flower's:

1. **Sepal Length** (cm)
   - Measure from base to tip
   - Usually 4.3-7.9 cm

2. **Sepal Width** (cm)
   - Measure at widest point
   - Usually 2.0-4.4 cm

3. **Petal Length** (cm)
   - Measure from base to tip
   - Usually 1.0-6.9 cm

4. **Petal Width** (cm)
   - Measure at widest point
   - Usually 0.1-2.5 cm
